BuddyNS Plesk Extension

Connect Plesk to BuddyNS!

This extension adds to plesk's custom-backend dns feature to create, update and remove zones as plesk does. buddyns.php is registered as a custom backend, and Plesk will pass the zone data to the script via stdin. API Calls to BuddyNS are made via php cURL.

After Installing, you'll still need to follow the instructions at BuddyNS AXFR Setup to allow buddyNS to transfer your zones.
